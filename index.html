1. Jenis-jenis Constructor di Dart

Constructor adalah fungsi khusus di dalam class yang digunakan untuk menginisialisasi atau membuat sebuah objek. Dart mendukung beberapa jenis constructor, seperti Default/Generative Constructor untuk pembuatan objek standar, dan Named Constructor untuk memberikan kejelasan konteks pembuatan objek yang berbeda.

2. Constant Constructor

Digunakan untuk membuat objek yang bersifat immutable (nilainya tidak bisa diubah setelah dibuat). Membantu meningkatkan efisiensi memori karena Dart akan menggunakan objek yang sama jika isinya identik.

class KategoriMateri {
  final String namaMateri;
  const KategoriMateri(this.namaMateri); // Constant Constructor
}

3. Constructor dengan Initializer List

Initializer list memungkinkan kita menginisialisasi variabel instans sebelum isi tubuh (body) constructor dijalankan. Sangat berguna untuk melakukan validasi atau manipulasi nilai properti awal.

Mahasiswa(this.nama, this.nilai) : status = (nilai >= 70) ? "Lulus" : "Remedial";

4. Pengantar Enkapsulasi (Encapsulation)

Enkapsulasi adalah konsep OOP yang digunakan untuk menyembunyikan data sensitif di dalam objek dari akses langsung luar kelas. Tujuannya adalah melindungi integritas data agar tidak diubah sembarangan.

5. Public vs Private di Dart

Tidak seperti bahasa lain yang menggunakan kata kunci public atau private, Dart menerapkan pembatasan hak akses menggunakan tanda garis bawah atau Underscore (_) di awal nama variabel atau metode.

String nama;     // Bersifat Public
double _nilai;   // Bersifat Private (Hanya bisa diakses di file yang sama)

6 & 7. Getter dan Setter

Untuk mengontrol interaksi terhadap variabel private, kita menggunakan fungsi Getter (untuk membaca nilai) dan Setter (untuk mengubah nilai dengan memberikan fungsi logika validasi tambahan).

// Getter
double get nilaiMahasiswa => _nilai;

// Setter dengan proteksi validasi
set nilaiMahasiswa(double nilaiBaru) {
  if (nilaiBaru >= 0 && nilaiBaru <= 100) {
    _nilai = nilaiBaru;
  }
}
